Revolve


"To learn how to develop vidoegames you need to learn how to develop boardgames"
Revolve was our first semester project. The Task was to develop a boardgame based on a videogame. to adapt the key elements of a videogame into a boardgame without breaking any rights/copyrights

I was mainly responsible for the ruleset, building prototypes, leading playtests, and collecting feedback to improve the gameplay.

Grimm and Grimmer

Grimm and Grimmer is an Point-and-Click Adventure referencing the Grimmbrothers

technical Difficulties

Bild: 3d-wirkendes Game, Bild: schrägaufnahme von Character der auf Plane steht

Our Goal was to create a 2.5d Point-and-Click Adventure. What seems to be 3d is actually a 2d plane the character is standing on. It was also very difficult to To implement/deal with the Movement System i had to manually bulid a Nav-Mesh and "block out" the walkable Ground. Bild: Boxen um die 2d Elemente, Bild: grünes Nav-Mesh

InteractionSystem

because most of the view-able objects dont exist "alone" / "seperately" from the background i had to map-out the interactable/viewable Objects so the Player is able to click on them.

Bild: markierte, jetzt clickbare Objekte, Bild: InteraktionsRad

Dialog System

Bild: von Dialog mit AntwortOptionen, Bild: Aufbau von Code

to make implementing and editing dialogs easier as well as make the project easy to translate and build in we imported them out of csv files and linked different slots inside the dialog.
